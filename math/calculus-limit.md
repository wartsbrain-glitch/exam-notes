# 极限与连续

## 核心概念关系

### 极限存在的充要条件

**函数极限**

$$\lim_{x \to x_0} f(x) = A \iff f(x_0^-) = f(x_0^+) = A$$

即：函数在某点极限存在的**充分必要条件**是左极限与右极限各自存在并且相等。

**数列极限（柯西收敛原理）**

数列 $\{x_n\}$ 收敛的**充分必要条件**：

$$\forall \varepsilon > 0, \exists N, \text{当 } m,n > N \text{ 时}, |x_n - x_m| < \varepsilon$$

> 有界是数列收敛的**必要条件**（非充分）；单调有界是数列收敛的**充分条件**（非必要）。

### 极限与无穷小的等价刻画

$$\lim f(x) = A \iff f(x) = A + \alpha, \text{其中 } \alpha \text{ 为无穷小}$$

这是函数具有极限 $A$ 的**充分必要条件**。

---

## 无穷小的比较

### 等价无穷小（$x \to 0$）

$$\sin x \sim x, \quad \tan x \sim x, \quad \arcsin x \sim x, \quad \arctan x \sim x$$

$$1 - \cos x \sim \frac{1}{2}x^2, \quad e^x - 1 \sim x, \quad \ln(1+x) \sim x$$

$$a^x - 1 \sim x\ln a, \quad (1+x)^\alpha - 1 \sim \alpha x$$

### 等价无穷小的充要条件

$$\beta \sim \alpha \iff \beta = \alpha + o(\alpha)$$

> **注意**：等价无穷小替换仅适用于**乘除**，加减运算中一般不用。

---

## 两个重要极限

$$\lim_{x \to 0} \frac{\sin x}{x} = 1$$

$$\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e$$

## 求极限方法速查

| 方法 | 适用场景 |
|------|----------|
| 等价无穷小替换 | $\frac{0}{0}$ 型，乘除结构 |
| 洛必达法则 | $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型 |
| 夹逼准则 | 难以直接求，可构造上下界 |
| 单调有界 | 递推数列 $x_{n+1} = f(x_n)$ |
| 凑重要极限 | $1^\infty$ 型 |

### 洛必达法则

若 $\lim_{x \to x_0} \frac{f(x)}{g(x)}$ 为 $\frac{0}{0}$ 或 $\frac{\infty}{\infty}$ 型，则

$$\lim_{x \to x_0} \frac{f(x)}{g(x)} = \lim_{x \to x_0} \frac{f'(x)}{g'(x)}$$

## 函数的连续性

### 连续的定义

$$f(x) \text{ 在 } x_0 \text{ 连续} \iff \lim_{x \to x_0} f(x) = f(x_0)$$

即：**极限值 = 函数值**。

### 连续的充要条件

$$f(x) \text{ 在 } x_0 \text{ 连续} \iff f(x_0^-) = f(x_0^+) = f(x_0)$$

### 闭区间连续函数的性质

- **有界性定理**：闭区间上的连续函数必有界
- **最值定理**：闭区间上的连续函数必能取到最大值和最小值
- **介值定理**：$f(a) \neq f(b)$，则对任意 $C$ 介于两者之间，存在 $\xi$ 使 $f(\xi)=C$
- **零点定理**：$f(a)f(b) < 0$，则存在 $\xi \in (a,b)$ 使 $f(\xi) = 0$

## 充分必要关系汇总

| 命题 | 条件类型 |
|------|----------|
| 极限存在 | $\iff$ 左极限 = 右极限 |
| 数列收敛 | $\iff$ 柯西条件 |
| 函数有界 | $\iff$ $|f(x)| \le M$ |
| 连续 | $\iff$ 左连续 + 右连续 = 函数值 |
| 数列有界 | 是收敛的**必要**条件 |
| 单调有界 | 是收敛的**充分**条件 |
