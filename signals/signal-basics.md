# 信号的分类与运算

## 信号的分类

### 按确定性

- **确定信号**：可以用明确的数学表达式描述
- **随机信号**：无法预测，只能用统计特性描述

### 按时间变量

- **连续时间信号** $x(t)$：时间变量 $t$ 连续取值
- **离散时间信号** $x[n]$：时间变量 $n$ 仅取整数值

### 按周期性

- **周期信号**：$x(t) = x(t + T)$，最小正周期为 $T$
- **非周期信号**：不存在满足上式的 $T$

### 按能量与功率

- **能量信号**：总能量有限，$0 < E < \infty$
  $$E = \int_{-\infty}^{+\infty} |x(t)|^2 dt$$
- **功率信号**：平均功率有限，$0 < P < \infty$
  $$P = \lim_{T \to \infty} \frac{1}{T} \int_{-T/2}^{T/2} |x(t)|^2 dt$$

## 基本运算

### 时移

$$x(t) \to x(t - t_0)$$

- $t_0 > 0$：右移（延迟）
- $t_0 < 0$：左移（超前）

### 尺度变换

$$x(t) \to x(at)$$

- $a > 1$：时间轴压缩
- $0 < a < 1$：时间轴扩展
- $a < 0$：反折+尺度变换

### 反折

$$x(t) \to x(-t)$$

以纵轴为对称轴翻转。

## 常用基本信号

### 单位阶跃信号

$$u(t) = \begin{cases} 1, & t > 0 \\ 0, & t < 0 \end{cases}$$

### 单位冲激信号 $\delta(t)$

**筛选性质**：

$$\int_{-\infty}^{+\infty} x(t) \delta(t - t_0) dt = x(t_0)$$

$$x(t) \delta(t - t_0) = x(t_0) \delta(t - t_0)$$

### 复指数信号

$$x(t) = e^{st} = e^{\sigma t} (\cos \omega t + j \sin \omega t)$$

其中 $s = \sigma + j\omega$。

---

**记忆碎片**：$\delta(t)$ 不是普通函数，是广义函数（分布），重点掌握其筛选性质和与 $u(t)$ 的关系 $\delta(t) = \frac{du(t)}{dt}$。
