# 系统的性质与分类

## 系统的分类

### 按信号类型

- **连续时间系统**：输入输出均为连续时间信号
- **离散时间系统**：输入输出均为离散时间信号
- **混合系统**：模数/数模转换系统

### 按系统性质

- **线性系统**：满足叠加性和齐次性
- **非线性系统**

### 按记忆性

- **无记忆系统**：输出仅取决于当前时刻输入
- **记忆系统**：输出与过去或未来的输入有关

## 线性时不变系统（LTI）

### 线性性（叠加性）

若

$$x_1(t) \to y_1(t), \quad x_2(t) \to y_2(t)$$

则

$$a x_1(t) + b x_2(t) \to a y_1(t) + b y_2(t)$$

### 时不变性

若 $x(t) \to y(t)$，则

$$x(t - t_0) \to y(t - t_0)$$

### 因果性

系统在 $t_0$ 时刻的输出只取决于 $t \le t_0$ 时刻的输入。

- **因果系统**：$h(t) = 0, \quad t < 0$
- **非因果系统**：物理不可实现

### 稳定性（BIBO 稳定）

有界输入产生有界输出。

对于 LTI 系统，稳定的充要条件：

$$\int_{-\infty}^{+\infty} |h(t)| dt < \infty$$

---

**记忆碎片**：考研中绝大多数系统都是**因果 LTI** 系统，分析时默认因果性，重点判断稳定性。
