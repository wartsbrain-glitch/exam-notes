# 图表作文

## 常用描述词汇

### 上升

- increase / rise / grow / soar / surge / climb
- an upward trend / a growing tendency

### 下降

- decrease / decline / fall / drop / plunge / slump
- a downward trend

### 波动/稳定

- fluctuate / remain stable / level off / stay constant

### 程度副词

- dramatically / sharply / steadily / slightly / gradually

## 常用句型

- The number of ... increased sharply from ... to ...
- There was a dramatic rise in ... during the period from ... to ...
- Compared with ..., ... witnessed a significant growth.

---

> 图表作文重点是"描述变化+分析原因+总结预测"。
